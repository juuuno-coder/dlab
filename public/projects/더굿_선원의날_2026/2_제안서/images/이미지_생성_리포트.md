## 스톡 이미지 — 미완료

스톡 이미지 다운로드는 서버가 로컬 파일시스템에 저장하는 방식이라
VM에서는 파일 경로 접근이 불가. 로컬 PowerShell에서 직접 실행 필요:

```powershell
# 예시: 스톡 이미지 다운로드
$headers = @{ Authorization = "Bearer YOUR_API_KEY"; "Content-Type" = "application/json" }
$body = '{"download_id":"pexels:35157037","resize_width":1920,"format":"png"}'
Invoke-RestMethod -Uri "http://localhost:3500/api/download-stock" -Method POST -Headers $headers -Body $body
```
