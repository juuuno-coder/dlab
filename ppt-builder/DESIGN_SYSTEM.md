## 5. 공통 패턴 & 규칙

### 5-1. 헤더 텍스트 형식

```
Part  {로마숫자} .  {섹션명}
```

예시: `Part  Ⅰ .  제안개요`, `Part  Ⅱ .  제안업체일반`

서브타이틀 형식:
```
{번호}.  {소제목}
```

예시: `2.  사업 이해 및 추진 방향`

### 5-2. 푸터 키워드 바 형식

모든 콘텐츠 페이지 하단에 공통:

- 배경: `#002B5B` (full width)
- 텍스트: 14pt Bold 흰색
- 패턴: `"{키워드1}({영문}) · {키워드2}({영문}) · {키워드3}({영문}) 전략을 통해…"`
- 위치: y=7.09" h=0.64"

### 5-3. 섹션 구분 설명문 하이라이트 규칙

네이비 배경 위 설명문에서 핵심 키워드만 `#FFD700` (골드) 처리:

```
일반 텍스트: #E5E7EB (연회색)
핵심 키워드: #FFD700 (골드)
```

예시: "성공적 개최를 위한 `기획 방향과 전략`을 제안합니다."

### 5-4. 구분선 규칙

| 유형 | 색상 | 두께 | 위치 |
|------|------|------|------|
| 헤더 하단 수평선 | #414042 | 1px | y=1.60" |
| 카드 하단 구분선 | #EEEEEE | 1px | 카드 하단 경계 |

## 6. PptxGenJS 빌더 변환 가이드

### 6-1. HTML → PptxGenJS 좌표 변환

```
HTML 기준: 1280 × 907 px
PPTX 기준: 11.69 × 8.27 인치

변환 공식:
  inch_x = (px_x / 1280) × 11.69
  inch_y = (px_y / 907) × 8.27

  또는 직접 XML EMU 값 사용:
  inch = emu / 914400
```

### 6-2. PptxGenJS 초기화 코드

```js
const pptxgen = require("pptxgenjs");

const pres = new pptxgen();
pres.defineLayout({ name: 'A4_LANDSCAPE', width: 11.69, height: 8.27 });
pres.layout = 'A4_LANDSCAPE';
pres.author = "PPT빌더";

// 디자인 시스템 상수
const C = {
  navy:    "002B5B",
  red:     "C8102E",
  gold:    "FFD700",
  dark:    "414042",
  black:   "1A1A1A",
  body:    "555555",
  caption: "888888",
  check:   "444444",
  bg:      "F8F9FA",
  numbg:   "F3F6F9",
  border:  "E5E7EB",
  white:   "FFFFFF",
  muted:   "E5E7EB",
  highlight: "FFD700"
};

const FONT = {
  sans: "Noto Sans KR",
  serif: "Noto Serif KR",
  kopub: "KoPub돋움체_Pro Light"
};

const SLIDE_W = 11.69;
const SLIDE_H = 8.27;
```

### 6-3. 공통 컴포넌트 함수

```js
// 페이지 번호
function addPageNumber(slide, num) {
  slide.addText(String(num), {
    x: 11.09, y: 7.73, w: 0.24, h: 0.16,
    fontSize: 12, color: C.dark, align: "center", valign: "middle"
  });
}

// 헤더 프레임 (콘텐츠 페이지용)
function addContentHeader(slide, { part, partTitle, subNum, subTitle }) {
  // Part 타이틀
  slide.addText(`Part  ${part} .  ${partTitle}`, {
    x: 0.78, y: 0.49, w: 5.65, h: 0.49,
    fontSize: 14, fontFace: FONT.sans, bold: true
  });
  // 서브타이틀
  slide.addText(`${subNum}.  ${subTitle}`, {
    x: 0.78, y: 1.26, w: 2.99, h: 0.25,
    fontSize: 14, fontFace: "Arial"
  });
  // 구분선
  slide.addShape("line", {
    x: 0.59, y: 1.60, w: 10.51, h: 0,
    line: { color: C.dark, width: 1 }
  });
}

// 푸터 키워드 바
function addFooterBar(slide, keywordText) {
  // 네이비 배경
  slide.addShape("rect", {
    x: 0, y: 5.92, w: SLIDE_W, h: 2.34,
    fill: { color: C.navy }
  });
  // 키워드 텍스트
  slide.addText(keywordText, {
    x: 0, y: 7.09, w: SLIDE_W, h: 0.64,
    fontSize: 14, fontFace: FONT.sans, bold: true,
    color: C.white, align: "center", valign: "middle"
  });
}
```

## 8. 색상 프로젝트 교체 가이드

다른 프로젝트(예: 봄꽃 전시회)에 적용할 때 교체할 색상 맵:

```json
{
  "선원의날": {
    "primary": "#002B5B",
    "accent1": "#C8102E",
    "accent2": "#FFD700",
    "mood": "navy_formal"
  },
  "봄꽃전시회": {
    "primary": "#???",
    "accent1": "#???",
    "accent2": "#???",
    "mood": "pink_nature"
  }
}
```

변경 시 `C.navy`, `C.red`, `C.gold` 상수만 교체하면 전체 톤앤매너가 바뀌는 구조.
