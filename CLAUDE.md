## 의미 단위 줄바꿈 규칙 (Semantic Line Breaks)

**모든 텍스트 콘텐츠에 적용되는 필수 규칙입니다.**

긴 텍스트는 사용자가 읽기 쉽도록 의미 단위로 줄바꿈을 적용합니다.

### 적용 기준
1. **한 줄에 15자 이상**인 description, subtitle 텍스트
2. **의미상 끊어 읽을 곳**에서 줄바꿈 (쉼표, 접속사 위치)
3. **대략 1/2 지점**에서 줄바꿈하여 균형감 유지
4. **최대 2줄 유지** (3줄 넘지 않게)

### 좋은 예시 ✅
```tsx
// ❌ 나쁜 예
description: '전문 메뉴얼과 체계적 시스템으로 신속하게 완벽한 칵테일 제공'

// ✅ 좋은 예
description: '전문 메뉴얼과 체계적 시스템으로\n신속하게 완벽한 칵테일 제공'
```

### JSX/TSX에서의 사용
```tsx
<p className="text-gray-400 leading-relaxed">
  전문 메뉴얼과 체계적 시스템으로<br />
  신속하게 완벽한 칵테일 제공
</p>
```

### 적용 효과
- 📖 **가독성 향상**: 눈의 이동 거리 최소화
- 🎯 **집중도 향상**: 의미 단위로 끊어 읽어 이해도 증가
- 💯 **전문성**: 세심한 타이포그래피로 브랜드 신뢰도 향상
- 📱 **모바일 최적화**: 작은 화면에서도 읽기 편함
